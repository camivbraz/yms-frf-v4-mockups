4.2.3.4 — STA Check on Geofence Entry

Detailed system check flow when a vehicle enters the Station Geofence

Vehicle enters Station Geofence
Step 1 — Arrival Type check
System reads Arrival Type from the driver's task
Field read from driver's task — path: [path to be defined]
Arrival Type — can be LH Inbound, LH Outbound, or other operation types. STA check only applies to LH Inbound and LH Outbound.
Step 2 — Arrival Type evaluation
Is the Arrival Type LH Inbound or LH Outbound?
STA check only applies to these two arrival types
YES — LH Inbound or LH Outbound
Step 3 — STA threshold check
Is an STA threshold configured at station level for this arrival type?
Reads from Vehicle Capacity Control — STA Threshold card
Field read from station config — Vehicle Capacity Control
STA LH Inbound or STA LH Outbound — value in minutes
Step 3b — LH Trip data retrieval
System reads scheduled arrival time from the LH Trip
Field read from LH Trip — path: LineHaul > Linehaul Trips
STA/STD — scheduled arrival time at the station · used to calculate the release time window
No threshold configured
STA check skipped
No restriction applied
→ 4.2.3.6
Entry site assignment
Threshold configured
Step 4 — Window calculation
Release time formula
Release time = Scheduled arrival − STA threshold
e.g. LH arrival 14:30 · STA = 60 min → Release time = 13:30
Step 5 — Arrival time evaluation
Current time ≥ Release time?
Is the vehicle arriving within the STA window?
YES — within window
Status: ASSIGNED
Proceeds to entry site check
→ 4.2.3.6
Entry site assignment
NO — arrived too early
Status: PENDING
Driver notified · system monitors until release time
→ 4.2.3.5
Early arrival monitoring
NO — other operation type
STA check skipped
Not applicable
→ 4.2.3.6
Entry site assignment
Driver status summary
Scenario STA check executed? Driver status Next step
Arrival Type is not LH Inbound or LH Outbound No — 4.2.3.6 Entry site assignment
Arrival Type is LH · no STA threshold configured No — 4.2.3.6 Entry site assignment
Arrival Type is LH · within STA window Yes ASSIGNED 4.2.3.6 Entry site assignment
